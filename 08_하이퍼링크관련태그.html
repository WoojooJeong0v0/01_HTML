a(anchor : 서로 다른 두 페이지를 연결) 태그

    - 현재 HTML 문서와 다른 HTML 문서를 연결 시켜 클릭 시 이동할 수 있게 만드는 태그

    - a태그로 감싸진 글/내용 == 하이퍼 텍스트

    - 하이퍼 텍스트는 다른 문서와 구분되는 HTML만의 핵심 기능(이었다가 word를 시작으로 퍼짐)

a 태그 테스트

    - 속성2가지
    1) href (Hyper text REFerence) : 연결할 주소를 작성하는 속성

    2) target = "_blank" : 새 탭으로 열기
  1. 글자 관련 태그
  2. 목록 관련 태그
  3. 연습 문제

페이지 내부에서 점프하기

    - 한 페이지 내에서 지정된 id 속성 값을 가진 요소로 이동
    * id (식별자) 속성 :
      요소를 구분하는 용도의 속성으로 한 HTML 문서 내에 중복되는 id가 존재하면 안 된다.
