p (Paragraph 문단) 태그

pre 태그처럼 형식이 유지되지는 않음.

다른 문단, 태그 사이에 여백이 생성된다.

글자에 적용되는 기타 태그들

b(bold) 태그 : 단순히 글자를 굵게 표시하는 태그

i(italic) 태그 : 단순히 글자를 기울여 표시하는 태그

strong 태그 : 글자(단어)를 강조하는 용도의 태그

em 태그 : 글자(문장)를 강조하는 용도의 태그

mark 태그 : 형광펜 효과를 추가하는 태그
u 태그 : 밑줄을 긋는 태그
s 태그 : 글자 가운데 선(취소선)을 긋는 태그, 잘못 작성됨
del 태그 : 글자 가운데 선(삭제선)을 긋는 태그, 이제 없음을 표시
small 태그 : 글자를 작게 표시하는 태그
